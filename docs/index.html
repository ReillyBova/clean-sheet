Snap a part on your phone — angled, glare-lit, curling off the table — and Clean Sheet lifts it flat into a crisp, correctly-sized PDF. Watch it work, one stage at a time.

Pipeline stage

The capture

A phone photo on a table — angled, unevenly lit, curling at the edges.

Runs offline, on your machine

A single Python CLI, flat_scan.py. Point it at a PDF or a whole folder of parts and it batches them — segmentation, Coons-patch dewarping, soft-gray ink, and optional staff-line straightening.

Built for real orchestra parts

Correct physical page sizes, duplex-friendly parity, and a booklet mode that detects the binding fold so a facing page never bleeds in.
